# あなたの相棒メモ（personal-memo）

シフトスケジューラーとは**完全に独立した個人用アプリ**です。あなた自身のこと
（型番・サイズ・好み・買ったもの＆感想・日々考えたこと）を気の向くままに貯めて、
いざという時にどのAIにも丸ごと引き継げるようにするためのもの。

- シフトスケジューラーのアプリ画面からは一切リンクされていません。
- ビルド不要・依存なしの**単一HTMLファイル**です。
- データはこのアプリを開いたブラウザの `localStorage`（キー `personal-memo-data`）に保存され、
  シフトスケジューラーのデータとは別物です。社内で見せるアプリには出てきません。

## 使い方

`personal-memo/index.html` をブラウザで開くだけ。

- **ダブルクリックで開く**（`file://` で動作）
- または、このフォルダだけを別の場所にコピーして使う
- スマホ等でも使いたい場合は、このファイルを**シフトスケジューラーとは別の静的ホスティング**に
  置いてください（同じ場所に置くと社内の人から見えてしまうので注意）。

## 3つのタブ

1. **🪪 プロフィール** … 型番・サイズ・好みなど「あなたの仕様」をカテゴリ付きで蓄積
2. **📝 日々の記録** … 考えたこと・買ったもの・感想を日付＋タグで記録
3. **🔄 引き継ぎ・バックアップ**
   - 全部を1枚の「私について（AI向け）」テキストにして**コピー / .md保存**
   - **JSON**で全データを書き出し／読み込み（端末間移行・バックアップ）
   - 他チャットの相談文を貼り付けて記録に取り込み

## 注意

ブラウザのデータを消す（キャッシュ削除等）と消えるので、大事な内容は時々
JSONで書き出してバックアップしてください。
